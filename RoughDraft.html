What is computer programming?

What is computer programming?

Computer Programming is the process of writing code to facilitate specific actions in a computer.

You can think of it like a set of instructions for a computer to follow to create and operate software or applications.

Since computers only know 1 language, binary, the programmer needs to write these instructions in certain programming languages.

History

Computing or calculating machines have been around long before modern computers.

Do you know what the first calculating machine humans ever used?

The Abacus dates back to 1100 B.C. A rectangular device with horizontal bars going across the middle, attached to these bars are beads that represent a certain unit while counting. If the bead is on one side, it represents a 0; on the other, it represents a 1. Which is very similar to binary code that computers use to function.

CSS

Cascading Style Sheets or CSS is the language developers use to style an HTML document.

CSS was formed by Hakon Wium Lie in 1994 while working alongside Tim Berners-Lee to start the formation of the World-Wide-Web.

CSS

In 1994 Hakon Wium Lie proposed the idea of adding a style sheet, like ones used for newspapers, to webpages.

With the use of the link tag, he proposed the idea of cascading, or being able to design more than 1 page with the same style.

In this proposal, he discussed ideas for properties that change fonts, font size, font weight, background colors, element alignment, margins between elements, and the length and width for the window.

JavaScript

JavaScript

JavaScript is a programming language used to make web pages interactive.

JavaScript can calculate, manipulate and validate data. It can also update and change both HTML and CSS.

JavaScript was invented by Brendan Eich in 1995 for Netscape 2, the first commercial web browser

JavaScript

In the early years of the World Wide Web, webpages lacked the capability for dynamic behavior after the page was loaded in the browser.

JavaScript allowed for both front-end and back-end programming, which allowed developers to build complex interactive websites and to connect servers to websites and web applications
